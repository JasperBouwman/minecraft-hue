# Minecraft Philips Hue

> A Minecraft plugin for Connecting to Philips Hue.


## Setup the Project

```
gradlew setupDecompWorkspace
```


### IntelliJ

1. Import the *build.gradle* file in IntelliJ.
2. Run the following command:

   ```
   gradlew genIntellijRuns
   ```


## Running

```
gradlew runClient
```
